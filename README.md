# Student Task Manager

A simple full-stack Flask web application with:

- User Registration
- User Login
- JWT Authentication
- Add Tasks
- View Tasks
- Delete Tasks
- MongoDB Database
- HTML/CSS/JavaScript Frontend

---

# Features

✅ JWT Authentication  
✅ Flask Backend  
✅ MongoDB Database  
✅ Task Management  
✅ Login & Register System

✅ Protected APIs  
✅ Responsive UI
